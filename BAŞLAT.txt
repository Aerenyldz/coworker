SORUN YAŞIYORSAN
════════════════════════════════════════════════════════════════

1. "Ollama sunucusu çalışmıyor" hatası?
   → Terminalde yeni pencere aç
   → Komut: ollama serve
   → run_tenra_v5.bat'ı tekrar çalıştır

2. "PySide6 hatası" veya paket kurulmadı?
   → Terminalde aç (.venv aktif ol)
   → Komut: pip install -r tenra_v5\requirements.txt

3. "Model yüklenmiyor" hatası?
   → Ollama'da model yükle: ollama pull qwen2.5:7b

4. Diğer sorunlar?
   → Terminalde çalıştır: python check_setup.py
   → Çıktı sorunun ne olduğunu gösterecek

AYARLAR
════════════════════════════════════════════════════════════════

✏️ Ayarları düzenlemek için: tenra_v5\config.py

  • Ollama URL: OLLAMA_URL = "http://localhost:11434/api"
